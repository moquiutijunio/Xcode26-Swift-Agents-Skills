# Xcode Intelligence Studies

A study project focused on exploring **Xcode Intelligence in Xcode 2026**, with hands-on experiments around **AI agents** in iOS development.

## About the project

This repository was created as a learning environment to explore how **Xcode Intelligence** can support code writing, feature evolution, and AI-assisted workflows within the Apple ecosystem.

As a starting point for these experiments, the project uses a **SwiftUI registration form**, which serves as the initial scenario for validations, testing, and future agent-based interactions.

## Goals

- Study the use of **Xcode Intelligence** in Xcode 2026
- Explore integration with **OpenAI**
- Learn practical applications of **AI Agents**
- Use **Swift Testing** with agent support
- Build a simple foundation for experimentation and continuous learning

## Technologies and concepts

- Xcode Intelligence
- AI Agents
- Swift Testing
- SwiftUI
- OpenAI Integration
